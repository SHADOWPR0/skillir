# Roadmap

## v0.1 — shipped foundation

- Markdown-to-Skill IR compiler
- capability-gap report, mock evidence, deterministic bundle, local registry
- neutral profiles, starter catalog, public schemas, safety boundaries

## v0.2 — document intelligence

- pinned offline PDF/OCR extraction and visual provenance
- interactive review, source change impact, demonstration-plan generator
- richer primitive and occupational coverage catalog

## v0.3 — interoperability

- adapter conformance suite
- target bundle exporters, dataset export, simulator integrations
- OCI registry publishing, cryptographic release attestations, private overlays

## v1.0 — trusted public protocol

- stable Skill IR and migration policy
- public evidence leaderboard and catalog site
- independent adapter implementations and reproducible external pilots
